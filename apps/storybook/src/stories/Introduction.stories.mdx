# iOS26 React Component Library

Welcome to the iOS26 React Component Library documentation. This library provides React components that replicate the iOS 26 design language for web applications.

## Getting Started

### Installation

```bash
npm install @ios26/ui
# or
pnpm add @ios26/ui
# or
yarn add @ios26/ui
```

### Basic Usage

```tsx
import { Button } from '@ios26/ui'
import '@ios26/ui/styles'

function App() {
  return <Button>Hello iOS26</Button>
}
```

## Design Philosophy

This library brings iOS 26's design language to the web, emphasizing:

- **Clarity**: Clear visual hierarchy and readable typography
- **Depth**: Subtle shadows and blur effects create depth
- **Deference**: Content takes priority over chrome
- **Accessibility**: WCAG 2.1 AA compliant components

## Component Structure

All components are built on:
- **React 18+**: Modern React features
- **shadcn/ui**: Component architecture
- **Radix UI**: Accessible primitives
- **Tailwind CSS**: Utility-first styling
